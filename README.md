## 📦 Installation & Deployment

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR-USERNAME/ayaan-portfolio.git
   ```

2. Open `index.html` in your browser

### Deploy to GitHub Pages
1. Create a new repository on GitHub named `ayaan-portfolio`

2. Push your code:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR-USERNAME/ayaan-portfolio.git
   git push -u origin main
   ```

3. Enable GitHub Pages:
   - Go to repository **Settings**
   - Navigate to **Pages** section
   - Under **Source**, select `main` branch
   - Click **Save**
   - Your site will be live at: `https://YOUR-USERNAME.github.io/ayaan-portfolio/`

## 🎨 Customization

### Colors
Edit the CSS variables in `style.css`:
```css
:root {
  --bg: #0a1628;          /* Dark navy background */
  --surface: #0f1d30;     /* Surface color */
  --accent: #00d4ff;      /* Electric blue */
  --text: #c5d1de;        /* Soft grey text */
  --muted: #6b7a8f;       /* Muted text */
}
```

### Content
- **About section**: Edit the story in `index.html`
- **Skills**: Update skill items in the skills section
- **Services**: Modify service cards
- **Contact info**: Update email and social links
